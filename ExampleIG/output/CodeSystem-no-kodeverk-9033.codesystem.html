CodeSystem: 9033 Slektskap mv. til pasient

Official URL: http://helsenorge.no/fhir/CodeSystem/no-kodeverk-9033.codesystem Version: 0.2.0
Active as of 2022-11-11 Responsible: Direktoratet for e-helse Computable Name: NoKodeverk9033
Other Identifiers: id: urn:oid:2.16.578.1.12.4.1.1.9033

Dette kodeverket inneholder koder som benyttes for å angi hvilket slektskap eller annet forhold , det er mellom pasienten og den pårørende.

This Code system is referenced in the content logical definition of the following value sets:

  • This CodeSystem is not used here; it may be used elsewhere (e.g. specifications and/or implementations that use this content)
